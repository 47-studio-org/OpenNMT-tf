opennmt.models.LanguageModelInputter¶

class opennmt.models.LanguageModelInputter(*args, **kwargs)[source]¶

A special inputter for language modeling.

This is a single word embedder that simply produces labels by shifting the input sequence.

Inherits from: opennmt.inputters.WordEmbedder
